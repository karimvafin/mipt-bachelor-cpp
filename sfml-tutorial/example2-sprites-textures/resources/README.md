# Ресурсы для примера 2

Поместите сюда ваше изображение для спрайта.

## Рекомендации

- Формат: PNG или JPG
- Размер: любой (рекомендуется 64x64 - 256x256 пикселей)
- Имя файла: `sprite.png`

## Где взять изображения?

- [OpenGameArt.org](https://opengameart.org/) - бесплатные игровые ресурсы
- [Pixabay](https://pixabay.com/) - бесплатные изображения
- Создайте свое изображение в любом графическом редакторе

## Альтернатива

Если изображение не найдено, программа создаст простой красный квадрат программно.
