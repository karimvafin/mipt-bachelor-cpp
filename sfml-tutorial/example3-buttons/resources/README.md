# Ресурсы для примера 3

## Шрифт (опционально)

Поместите сюда файл шрифта для отображения текста на кнопках.

### Рекомендации

- Формат: TTF или OTF
- Имя файла: `font.ttf`
- Рекомендуемые бесплатные шрифты:
  - [Google Fonts](https://fonts.google.com/) - огромная коллекция бесплатных шрифтов
  - [Font Squirrel](https://www.fontsquirrel.com/) - качественные бесплатные шрифты

### Популярные шрифты для кнопок

- **Roboto** - современный, читаемый
- **Open Sans** - универсальный
- **Lato** - элегантный

### Без шрифта

Если шрифт не загружен, кнопки будут работать, но без текста (только цветные прямоугольники). Это нормально для изучения функциональности!

### Системные шрифты

На macOS можно использовать системные шрифты:
- `/System/Library/Fonts/Helvetica.ttc`
- `/Library/Fonts/Arial.ttf`

Программа автоматически попытается найти шрифт в стандартных местах.
